In this example, we create a div element with a class of clip-path-example. We set the width and height to 200 pixels, and the background color to red. Then, we use the clip-path property to create a polygon shape. The polygon() function takes a series of x and y coordinates as arguments, separated by spaces. Each pair of coordinates represents a point on the polygon. In this example, we use four points to create a diamond shape. The 50% 0 coordinate represents the top point of the diamond, 100% 50% represents the right point, 50% 100% represents the bottom point, and 0 50% represents the left point. The -webkit-clip-path property is included for compatibility with older versions of WebKit-based browsers, such as Safari. When you view the HTML file in a web browser, you should see a red diamond-shaped element with a white background (assuming no other styles are applied).
